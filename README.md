# Iris Flower Classification 🌸

## Project Overview

Iris Flower Classification is a Machine Learning project that predicts the species of iris flowers based on their measurements.

The model learns from flower measurements and classifies flowers into three categories:

- Iris-setosa
- Iris-versicolor
- Iris-virginica


## Objective

To build a Machine Learning classification model that can identify iris flower species using:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width


## Dataset

Dataset Name:
Iris.csv

The dataset contains:

- 150 flower samples
- 4 numerical features
- 3 target classes


Features:

1. Sepal Length
2. Sepal Width
3. Petal Length
4. Petal Width


Target:

Species


## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn


## Machine Learning Algorithm

Algorithm Used:

Random Forest Classifier


## Project Workflow

1. Import required libraries

2. Load Iris dataset

3. Perform Exploratory Data Analysis

4. Visualize dataset

5. Split data into training and testing sets

6. Train Machine Learning model

7. Test model performance

8. Predict flower species


## Model Accuracy

The Random Forest Classifier achieved approximately:
